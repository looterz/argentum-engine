# Top Draft Sets to Implement

Community consensus top 10 draft formats, ranked by how frequently they appear in "best draft format" discussions.

## Tier 1 — Universal Picks

1. **Innistrad (INN)** — Tribal archetypes (Zombies, Vampires, Werewolves, Humans), flashback, double-faced cards. Spider Spawning kept the format evolving. Self-correcting — overdrafted archetypes opened up underdrafted ones.

2. **Rise of the Eldrazi (ROE)** — Slow, grindy, ramp-focused. Wall decks with Vent Sentinel, token strategies, levelers. Complete departure from hyper-aggressive Zendikar/Worldwake.

## Tier 2 — Widely Acclaimed

3. **Ravnica: City of Guilds block (RAV/GPT/DIS)** — Ten guild archetypes with excellent color balance. Gold cards pulled you into lanes without feeling forced, rewarded reading signals.

4. **Khans of Tarkir (KTK)** — *Already in progress.* Five wedge-color clans, morph as universal bluffing mechanic, good three-color fixing.

5. **Dominaria (DOM)** — Legendaries-matter, slow interactive games, common removal answered bomb rares. Kicker for late-game flexibility, historic mechanic tied the set together.

6. **Time Spiral block (TSP/PLC/FUT)** — Kitchen sink of returning mechanics, absurd number of viable archetypes. Timeshifted cards added wild variance. Rewarding for enfranchised players.

## Tier 3 — Modern Era Standouts

7. **Modern Horizons 2 (MH2)** — High power level done right. Deep archetypes, strong commons, tons of build-around potential.

8. **Modern Horizons (MH1)** — Snow matters, Ninjas, Slivers, Changelings. Throwback archetypes that played surprisingly well together.

9. **Amonkhet block (AKH/HOU)** — Aggressive but balanced. Exert, cycling, embalm all played great in limited.

10. **Invasion block (INV/PLS/APC)** — First truly great draft format in MTG history. Five-color goodstuff viable, kicker excellent, Apocalypse's enemy-color pivot rewarded flexible drafters.

## Implementation Notes

- Khans of Tarkir is already partially implemented (see `backlog/sets/khans-of-tarkir/`)
- Onslaught block (Onslaught, Legions, Scourge) is also in progress
- Sets requiring new mechanics (transform/DFCs for Innistrad, level up for ROE, etc.) will need engine work before card implementation
